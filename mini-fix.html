Fix the Mac Mini

You'll Screen Share into the Mini from your Air, then run one command. ~3 minutes.

1

Connect to the Mini via Screen Sharing

On your Air, open Finder. In the menu bar, click Go > Connect to Server (or press Command + K).

Type this and click Connect:

vnc://Garens-Mac-mini.local

If it asks for a name and password:

Username garen
Password GravelKing2026!

You should now see the Mini's desktop in a window on your Air.

If it says it can't connect, try this address instead: vnc://192.168.1.167
2

Open Terminal on the Mini

Inside the Screen Sharing window (you're now controlling the Mini), press Command + Space, type Terminal, hit Enter.

3

Paste this command

Copy this from your Air, then paste it into the Terminal inside the Screen Sharing window. Hit Enter.

tailscale up
4

It might ask for a password

If it says "Password:", type this and hit Enter. Nothing will show while you type — that's normal.

Password GravelKing2026!
5

Open the link it gives you

It will print a URL that starts with https://login.tailscale.com/...

Hold Command and click the link in the Mini's Terminal to open it in the Mini's browser. Or copy it and paste it into Safari on your Air — either works.

On the webpage, click Connect or Authorize.

If it says "Success" or just sits there with no URL, that's fine — it already connected. Skip to step 6.
6

Text Garen "done"

You can close the Screen Sharing window. That's it!

Thank you! This reconnects the Mini so Garen can reach it from anywhere.
